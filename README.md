## 🚀 Featured Project 1: Amazon Sales Dashboard 2024

### 🔍 Live Preview
<p align="center">
  <img src="Amazon Dashboard- Project 1.jpg" alt="Amazon Sales Dashboard 2024" width="100%" />
</p>

### 📝 Project Objective
The goal of this project was to analyze multi-city retail performance for Amazon sales in 2024. The dashboard provides comprehensive insights into sales distributions, target achievement metrics, product performance, and geographical breakdowns to assist stakeholders in identifying key growth regions and optimization points.

### ⚡ Key Features & Insights Built:
* **Dynamic KPI Metrics:** Tracked **Total Sales (76,505)**, **Total Units Sold (2,501)**, and geographic reach across **23 Cities**.
* **Target vs Achievement Analytics:** Designed custom donut charts tracking performance boundaries (`19%` Sales vs `81%` Not Achieved, and `91%` Units vs `9%` Not Achieved).
* **Geographical Mapping:** Integrated an interactive **City v/s Sales geographic map of India** paired with trend lines for rapid regional benchmarking.
* **Product Performance Tracking:** Built horizontal rank bars charting exact distributions for items like *Nestle Toffee Crumble*, *Milky Bar Sticks*, etc.
* **Full Interactivity:** Embedded synchronized custom **Slicers (City & Products)** allowing instant cross-filtering across all visual panels.
